关于搭建博客时遇到的一些错误

waterthecat · 2021-8-13 · 次阅读

关于搭建博客时遇到的一些错误

说在前面

​ 小站是基于HEXO并套用了Sakura主题构建的，在建站时遇到了许多困难（本质上是因为博主个人的编程经验约等于0造成的），不过经过几天的不断试错总算是勉强解决了大部分问题，所以通过记录下这些问题方便日后维护。同时希望能和各位交流交流心得，帮助其他同我遇到一样问题的博主，但无奈博主编程经验实在有限，文章内各种方法可能并不那么有效…

关于归档和清单的子项跳转到404界面这件事

在主题文件夹下languages/zh-cn.yml等文件中增加了相关子页面的信息仍然会跳转到404界面

只是还没有添加相应tag的博客罢了（悲）

关于aplayer播放器无法显示的二三事

以下错误均在使用网易云音乐的基础上发生，使用其它网站的歌曲/歌单的错误解决方法暂时不清楚

查看是否在主题文件夹下的_config.yml或根目录/course/music/index.md中调整了歌单/歌曲的id

查看是否将vip/失效或者其它限制播放的歌曲放入目标歌单中

查看是否将歌单设置为“公开”而非“私密”

查看目标歌单是否在电脑的网易云音乐app中播放过（我栽在这了）

详细说一下最后一点：就是下载了网易云音乐的电脑端app后播放过目标歌单的歌曲后aplayer就不能在博客中使用这个歌单，解决方法很简单，就是再建立一个空歌单把选定的歌曲再加入并且不要再在app中播放这个歌单。

关于iframe播放器无法加载这件事

同上，只限网易云音乐的iframe插件

这个问题没办法完美解决

实践下来使用第二种大小的插件并且在网易云的代码中插入http:而非https:

如下

<iframe frameborder="no" border="0" marginwidth="0" marginheight="0" width=330 height=110 src="//music.163.com/outchain/player?type=0&id=6913657188&auto=1&height=90"></iframe>

在//music.163.com/前方加入http:

这两个方法比较玄学，慎用

关于startdash中三个图片替换了却不能加载这件事

可能是因为更换的图片较大导致，可以看到自带的图片大小都仅有几十k

接下来是博主暂时无法解决的一些问题

希望知道解决办法的博主可以和水猫好好蕉流蕉流♂请使用QQ和博主取得联系，欧内该

关于使用Typora+Picgo上传文章图片到图床的那些事

使用Typora+picgo将文章内的图片自动上传到图床后再在cdn文件夹中使用

$ git status，$ git add .,$ git commit -m'****',$ git push

这些命令时会出现有关url的error

关于导航栏中“关于”的子项“关于我”的内容修改那些事

在Sakura/source/js/botui.js中仅能修改对话框中内容，但是无法修改对话框上方的“樱花庄的haojun”，“真白”以及“mashiro”等文本。博主尝试在根目录\source\about\index.md中进行修改，但发现以下注释

!– 因为vue和botui更新导至bug,现将对话移至js下的botui中配置 –

博主尝试在这附近

<div class="entry-content">
<div class="moe-mashiro" style="text-align:center; font-size: 50px; 
......

进行修改，可以改变对话框上方的多处文本，但确实会引起bug使对话框内内容无法显示。

关于赞赏界面内容无法删除那些事

文章末尾的赞赏界面内有支付宝和微信两个赞助二维码，博主在网上实在（懒）没找到应该在哪个文档删去内容，只能修改主题页下的_config来替换掉捐助的二维码图片了。关于这一项，博主深知自己的博客还不具备让人打赏的价值，所以在日后可能会考虑关闭所有赞赏入口。

关于首页视频无法播放这件事

博主上传了一个大小将近19m的视频且cdn路径有效，但一直显示加载中。

其它大佬说这个视频不应超过20m，也许是因为博主的视频比较大了，日后我会更换较小版本的视频再次尝试是否可以播放。

（2021.8.21更新：已经解决，在url处不应该填写完整地址，应填写到目标视频的文件夹处，在name处填写文件名）

关于改变窗口分辨率这件事

博主在本地测试时使用F12并拖动控制台边框改变大小时偶然发现的一个小bug

拖动的同时博客网页会自动适应分辨率的改变，但是多拖动几次（频繁改变窗口分辨率）网站会卡死，只能拖动无法点击。

这个应该没什么意义，没有谁会无聊到拖控制台玩吧哈哈
